# PinHan 扩展词库目录

此目录用于放置补充词表，扩展输入法的词汇覆盖率。

## 格式说明

每个文件为 `.txt` 格式，内容为一行一个词条，格式为：

```
词语1 频率1
词语2 频率2
...
```

频率为整数或浮点数，越高优先级越高。如无频率，默认为 1。

## 示例词表

- `hotwords.txt` - 网络热词、常用词补充
- `medical_terms.txt` - 医学专业术语
- `brand_names.txt` - 品牌和公司名称
- `custom.txt` - 用户自定义词表

## 添加新词表

1. 在此目录创建 `.txt` 文件（如 `my_vocab.txt`）
2. 按上述格式添加词语和频率
3. 运行 `python scripts/build_dict.py` 重新构建词典
4. 新词表会自动加载，优先级为 40（高于第三方词库）

## 优先级说明

- SUBTLEX-CH: 50（口语核心）
- extensions（本目录）: 40（扩展词库）
- sources: 30（第三方词库）
- CC-CEDICT: 10（兜底）
